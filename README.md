# Simulador de Conta Bancária 💰

Este é um dos meus primeiros projetos em Java, focado no domínio de estruturas de controlo e fluxo de dados. O sistema simula o funcionamento básico de um terminal bancário, permitindo interações reais de consulta e movimentação financeira via consola.

### ✨ Funcionalidades

* **Consulta de Perfil:** Exibição detalhada do titular, tipo de conta e saldo inicial configurado.
* **Menu Interativo:** Interface amigável para navegação entre diferentes operações bancárias.
* **Depósitos e Recebimentos:** Atualização instantânea do saldo após a entrada de novos valores.
* **Sistema de Transferência:** Permite o envio de valores com validação lógica (impede a saída de valores superiores ao saldo disponível).
* **Controlo de Fluxo:** Utilização de loops `while` e condicionais `if/else` para garantir a continuidade da execução até que o utilizador decida sair.

### 🛠️ Tecnologias e Conceitos Aplicados

* **Java SE:** Linguagem base do projeto.
* **Scanner:** Manipulação de entrada de dados do utilizador.
* **Lógica de Programação:** Implementação de regras de negócio para transações financeiras.
* **Estruturas de Repetição:** Gestão de menus dinâmicos.
